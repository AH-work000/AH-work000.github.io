Instructions: Fire Particle Curl Animated Mouse Pointer

Input Controls

  • Move Mouse --> Increase the number of particles in the fire paricle curl
  • Drag Mouse --> Increase the size of the particles
  • Double Click --> Reset the fire particles curl to default

Fire Curl Mouse Colours

  • Press Keyboard Keys to:
  • R = Red
  • G = Green
  • B = Blue
  • O = Orange
  • P = Purple
  • T = Trurquoise
